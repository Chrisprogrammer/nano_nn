# nano_nn

nano_nn is a modern single-header neural network library written in pure C99.

Designed to be lightweight, portable, and easy to integrate, nano_nn provides the building blocks required for training and deploying feed-forward neural networks without external dependencies.

Simply:

#define NANO_NN_IMPLEMENTATION
#include "nano_nn.h"

and start building neural networks.

No build system.
No runtime dependencies.
Just C.
